## **Problem Statement**
Users struggle with manual calendar management, lacking intelligent event organization, priority insights, and clear schedule visualization, leading to inefficient time management and missed appointments.

## **Scope of the Project**
- Single-user calendar application
- Event management with basic ML predictions
- Local JSON data storage
- Console-based interface
- Rule-based classification (no complex ML models)

## **Target Users**
- Students managing academic schedules
- Professionals organizing work meetings
- Individuals tracking personal appointments
- Anyone needing basic calendar organization

## **High-Level Features**
- **Event Management**: Create, view, update, delete events
- **AI Predictions**: Automatic event categorization and priority assessment
- **Calendar Visualization**: Monthly and daily schedule views
- **Data Persistence**: Automatic saving to JSON file
- **Input Validation**: Error handling for dates and times
- **Smart Suggestions**: ML-powered category and priority recommendations
